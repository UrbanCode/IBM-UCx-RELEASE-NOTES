7.2.1.1

IBM UrbanCode Deploy version 7.2.1.1 is a maintenance release and includes security related fixes. This release is recommended for all customers currently on the UrbanCode Deploy 7.2.1.x stream. The following updates and enhancements are included in this release:

  • A new agent script that sets the ANT_HOME and GROOVY_HOME variables.

    The tool_env script is added to the agent directory on upgrading to the UrbanCode Deploy V7.2.1.1. Agent startup tool uses this script to set and reference the environment variables appropriately.
