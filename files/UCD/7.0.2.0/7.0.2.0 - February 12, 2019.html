7.0.2.0 - February 12, 2019

Safe Edit

UrbanCode Deploy 7.0.2.0 announces Safe Edit functionality. Safe Edit allows users to add change management controls to automation design. Now users can create draft processes which can be used to test a processes functionality before promoting the draft to an official version. Optionally, users can require an approval process as part of the promotion process. For more information, check out our Safe Edit Overview video.

Java 11

UrbanCode Deploy 7.0.2.0 now supports Java 11. As part of this support, we have also upgraded our Groovy libraries to 2.4.15
