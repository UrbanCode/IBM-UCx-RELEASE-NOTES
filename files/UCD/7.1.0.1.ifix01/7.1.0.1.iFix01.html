7.1.0.1.iFix01

This release replaces the Urbancode Deploy version 7.1.0.1 release. It includes all updates made in the 7.1.0.1 release. Version 7.1.0.1 has been removed and not available to download. If you have installed version 7.1.0.1, install this iFix to prevent potential problems. The following updates and enhancements are included.

Getting Started


For supported platforms and requirements, see the reports that can be dynamically generated using the Software Product Compatibility Reports (SPCR) tool. As you plan your production topology, review the installation guide .
