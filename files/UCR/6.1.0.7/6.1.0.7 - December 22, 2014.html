6.1.0.7 - December 22, 2014

This release of IBM UrbanCode Release is a maintenance release containing bug fixes. This release is recommended for all customers.

Critical Bug Fix

This fixpack addresses an issue in 6.1.0.6 upgrades that causes all applications to be associated with all releases. Upgrading to this fixpack directly from 6.1.0.6 will reverse the effect, resolving the issue.

Release Summary


  • Critical data model update

Release Notes


For information on documentation and support resources, software and hardware requirements and installation steps, see the Getting Started page.

Installation Notes

The APAR fixed in this release solves an association issue between applications and releases. It is recommended that users on 6.1.0.6 upgrade to this fixpack.

Behavioral Changes

There are no known behavioral changes.

Fixes in this Release

A cumulative list of fixes in this release, and any future fix packs will be available.

APAR Description
— This version of IBM UrbanCode Release addresses an issue found after upgrading to version 6.1.0.6. Upgrading to version 6.1.0.6 caused all applications to be associated with all releases. Upgrading to version 6.1.0.7 directly from version 6.1.0.6 reverses this effect and resolves the issue.
