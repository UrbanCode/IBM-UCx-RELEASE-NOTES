2.5.1.iFix01

This release replaces the IBM Urbancode Velocity version 2.5.1 release. It includes all updates made in the 2.5.1 release. Version 2.5.1 has been removed and not available to download. If you have installed version 2.5.1, install this iFix to prevent potential issues. The following updates and enhancements are included.

2.5.1.iFix01

Fixed security vulnerability for Apache Log4j 2 / Log4Shell.

Insights update

On the Insights page, when you click the Reports tab, available reports include the following:
  • Pipeline snapshot
  • Deployment audit
  • Deployment duration (Beta)

Security

  • The 2.5.1.ifix01 addresses the issue of the security vulnerability to CVE-2021-44228, Apache Log4j 2 / Log4Shell.
  • Fixed security vulnerabilities.
